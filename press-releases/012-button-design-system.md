# Button Design System

**Date:** 2026-03-27

## What's New

- **Two-tier button design** — All buttons now follow a consistent pattern used across the monorepo:
  - **Object Create buttons** — Pastel-coloured background with a `+` prefix (e.g. `+ Event`, `+ Domain`, `+ Concept`). Used when adding new entities.
  - **Canvas Action buttons** — White background with a coloured border and coloured text (e.g. `New Matrix`, `Delete`, `Export JSON`, `Import JSON`). Used for actions on existing things.

- **Consistent across modes** — The same button styling applies in both the standalone app and the embedded Context Plane version.
