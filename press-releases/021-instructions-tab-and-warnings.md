# Instructions Tab, Missing-★ Warning, and Grey Domain Ticks

**Date:** 2026-04-17

## What's New

- **Instructions tab** — A second Tier 3 tab next to **Matrix** holds the in-app guide. Covers the example matrices, how to build your own, how to use the export buttons, and how to drive the matrix from Claude (Claude Code or Claude Chat).
- **Missing-★ warning indicator** — Events that have no driving Concept (no ★ in any cell of the row) show an amber warning marker (!) next to their concept count. Quick visual cue for events that are still missing their "this is the thing being created" anchor.
- **Grey domain ticks vs amber concept stars** — Domain ✓ marks render in a neutral grey while Concept ★ marks render in amber, making the two axes immediately distinguishable on a busy matrix.
- **W-type fix** — Cleaned up an edge case in W-type assignment when adding a concept that already exists with a different W value.
