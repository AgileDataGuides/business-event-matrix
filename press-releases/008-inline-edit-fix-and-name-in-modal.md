# Inline Edit Fix & Name in Edit Modal

**Date:** 2026-03-24

## What's New

- **Inline Edit Fix** — Fixed a bug where double-clicking to edit a domain or concept name inline caused the input field to shrink to 1 character wide. The field now sizes to fit the existing name.

- **Name in Edit Modal** — The "Edit details" popup now includes a Name field at the top, so you can rename domains and concepts from the modal as well as inline.
