# Domain Descriptions & Hover Tooltips

**Date:** 2026-03-24

## What's New

- **Domain Descriptions** — Domains now have a description field. Hover over any domain column header to see a tooltip popup showing the domain name and its description.

- **Hover Tooltip** — A light-themed popup appears when hovering over domain headers, with an "Edit description" link that opens a modal dialog to edit the description. The tooltip stays visible as you move your mouse to click the link.

- **Edit Modal** — A clean modal dialog for editing domain descriptions with a textarea and Save/Cancel buttons.
