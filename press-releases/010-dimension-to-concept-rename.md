# Dimension to Concept Rename

**Date:** 2026-03-27

## What's New

- **"Dimension" renamed to "Concept"** — All UI labels, column headers, buttons, and tooltips now use "Concept" instead of "Dimension" for clarity and alignment with business modelling terminology.

- **Internal types unchanged** — The underlying TypeScript types remain `Dimension` for backwards compatibility with existing saved data and adapters.

- **W's groupings preserved** — Concepts are still organised by W's (Who / What / When / Where / Why / How / How Many) with the same pastel colour coding.
