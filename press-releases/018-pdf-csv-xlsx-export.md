# PDF, CSV, and Excel Export

**Date:** 2026-04-04

## What's New

- **Export PDF** — Generate a print-ready PDF report of the matrix with domain-grouped columns. Available in both the standalone app and the Context Plane.
- **Export CSV** — Flat CSV of events with their domain and concept marks. One row per event, columns for Event Name + Description + each Domain + each Concept (grouped by W-type).
- **Export Excel** — XLSX workbook with three sheets: Matrix (the same shape as the CSV), Domains (name / description / owner / aliases), and Concepts (name / W-type / description / aliases).
- **Domains sort alphabetically on add** — New domains drop straight into their alphabetical slot on the matrix instead of appearing at the end.
