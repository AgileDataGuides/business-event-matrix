# Business Event Matrix Now Sorts Events and Concepts Alphabetically

**24 March 2026**

AgileDataGuides today released alphabetical sorting for events and concepts in the Business Event Matrix, so newly added items land in a predictable position instead of at the end of the list.

## The Problem

When users added a new event or concept, the app appended it to the end. After a few additions the matrix was in random insertion order, making it slow to find anything. Users were manually reordering rows and columns to keep things tidy — wasted clicks every time.

## The Solution

Events and concepts now sort alphabetically by name. Adding a new event drops it into the correct alphabetical row in the matrix; adding a new concept slots it into the correct alphabetical position within its W's group (Who / What / Where / When / Why / How / How Many).

## Key Benefits

- **Predictable ordering** — find any event or concept by scanning the alphabet
- **Zero-effort tidiness** — no manual reordering after every addition
- **W's grouping preserved** — concepts still cluster by their W's category; sorting happens within the cluster
- **Faster scanning** — large matrices are dramatically easier to read at a glance

Alphabetical sorting is available now in the latest release.
