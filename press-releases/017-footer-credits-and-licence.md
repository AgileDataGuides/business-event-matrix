# Business Event Matrix Adds Footer Credits and Open Licence

**27 March 2026**

AgileDataGuides today released a footer for the Business Event Matrix that credits the AgileDataGuides Pattern Template family and exposes the open licence terms users were asking for.

## The Problem

People kept asking two questions: "what's this part of?" and "can I use this at work?" The first because the app's relationship to the wider AgileDataGuides Pattern Template family wasn't visible inside the app itself — only on the GitHub repo. The second because nobody could find the licence without opening the README.

## The Solution

The standalone app now displays a footer at the bottom of every page with two pieces of information:

1. **AgileDataGuides Pattern Template attribution** — links to [agiledataguides.com](https://agiledataguides.com), placing the Business Event Matrix in context as one of a family of pattern template apps.
2. **CC BY-SA 4.0 licence notice** — links to the full Creative Commons Attribution-ShareAlike 4.0 International licence text.

## Key Benefits

- **Visible attribution** — the AgileDataGuides relationship is no longer hidden in the README
- **Visible licence** — users can confirm "yes, I can use this at work" without leaving the app
- **Friendly licence** — CC BY-SA 4.0 means free use, modification, and redistribution as long as derivative work shares the same licence

The footer is available now in the latest release.
