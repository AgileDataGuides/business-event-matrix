# Model to Matrix Rename

**Date:** 2026-03-27

## What's New

- **"Model" renamed to "Matrix"** — UI actions now read "New Matrix", "Delete", and "Switch matrix" instead of the previous model-based labels.

- **Custom matrix switcher dropdown** — The old `<select>` element has been replaced with a styled dropdown that shows the current matrix name, a "Switch matrix" subtitle, and a chevron. Active item is highlighted; click-outside-to-close behaviour included.
