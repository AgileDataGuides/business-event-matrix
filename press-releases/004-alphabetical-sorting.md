# Alphabetical Sorting for Events & Concepts

**Date:** 2026-03-24

## What's New

- **Alphabetical Concepts** — Concepts are now sorted alphabetically by name within each W's group (Who, What, Where, When, How). Adding a new concept automatically places it in the correct alphabetical position.

- **Alphabetical Events** — Events are now sorted alphabetically by name. Adding a new event automatically places it in the correct alphabetical position in the matrix.
