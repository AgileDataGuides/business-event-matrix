# Initial Event Matrix App

**Date:** 2026-03-24

## What's New

- **Event Matrix** — A web-based Event Matrix tool for business domain modelling, inspired by Lawrence Corr's BEAM "Who does What" pattern. Define business events (Subject Verb Object) and map them against concepts categorised by W's (Who, What, Where, When, How).

- **Concepts by W's** — Concepts (dimensions) are grouped into columns by their W's category (Who, What, Where, When, How), each with its own pastel colour coding.

- **Domains** — Define business domains (e.g. Sales, Operations) and map events to domains using star/check marks.

- **Interactive Matrix** — Click cells to toggle marks (empty → check → star → empty). Double-click event names, concept names, or domain names to edit inline.

- **Multiple Models** — Create, switch between, and delete multiple domain models. Includes a pre-built Ice Cream Shop example.

- **Import/Export** — Export models as JSON files and import them back.

- **Pastel Design** — Clean, light design inspired by the Context Plane app, using pastel backgrounds with coloured text and subtle borders throughout.
