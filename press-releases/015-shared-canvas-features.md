# Shared Canvas Features

**Date:** 2026-03-27

## What's New

- **Inline rename** — Double-click on event, concept, or domain names to rename them directly in the matrix. Ported to the shared `BusinessEventMatrixLayout` canvas component.

- **Hover tooltip** — Hovering over concepts and domains shows a rich tooltip with description, aliases, owner, and notes. Now part of the shared canvas.

- **Edit modal** — Click-to-edit modals for domains and concepts with all fields (name, description, aliases, owner, notes). Also moved into the shared canvas.

- **Identical interaction in both modes** — Standalone and Context Plane versions now share the exact same interaction patterns via the single-source canvas component.
