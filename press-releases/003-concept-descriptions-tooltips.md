# Concept Descriptions & Hover Tooltips

**Date:** 2026-03-24

## What's New

- **Concept Descriptions** — Concepts now have a description field, just like domains. Hover over any concept column header to see a tooltip with the concept name, description, and W's colour coding.

- **Unified Tooltip** — Both domain and concept tooltips share the same light-themed popup design. The border and title colour matches the item's category (violet for domains, green/blue/etc for concept W'ss).

- **Unified Edit Modal** — A single modal dialog handles editing details for both domains and concepts, with the header dynamically showing "Domain Details" or "Concept Details".
