# Auto-Detecting Import

**Date:** 2026-03-27

## What's New

- **Export Graph / Import Graph buttons removed** — The separate graph-format import/export buttons have been removed from the toolbar.

- **Single "Import JSON" button** — One import button now auto-detects the incoming format. It distinguishes between Context Plane graph format (nodes + links) and native BEM JSON, then applies the correct importer automatically.
