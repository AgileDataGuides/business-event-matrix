# Business Event Matrix Adds an Instructions Tab

**17 April 2026**

AgileDataGuides today released an Instructions tab built into the Business Event Matrix, giving new users an in-app guide to the matrix without sending them to a separate README or wiki page.

## The Problem

New users opened the app and immediately had questions. What's a Concept? What does the star mean? How do I create a new matrix? The README answered those — but the README was on GitHub, in a separate tab, and most users never saw it. The "click around and see what happens" approach worked for some, but lost others before they got their first matrix built.

## The Solution

A new **Instructions** tab now sits alongside **Matrix** at the bottom of the header (Tier 3). It contains a structured guide:

- **Quick Start** — describes the three example matrices that ship with the app and walks through clicking around the matrix
- **Build Your Own** — step-by-step for creating a matrix from scratch
- **No Claude Code? Use Claude Chat** — for users who don't have Claude Code installed, how to use the app with claude.ai instead
- **Managing Your Matrices** — switching, importing, exporting, all four export formats explained
- **Security and Data Storage notes** — the local-only commitment, where files live

## Key Benefits

- **Onboarding inside the app** — no need to leave for a README
- **Always up to date** — ships with the app, gets updated alongside features
- **Three example walkthroughs** — Ice Cream Shop, Lawrence Corr BEAM, SaaS Revenue Events
- **Both Claude Code and Claude Chat** — instructions for both flows, so users without Claude Code aren't stuck

The Instructions tab is available now in the latest release.
